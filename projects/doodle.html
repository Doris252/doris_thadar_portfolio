About the project

I built a fully-featured drawing application in C# using Windows Forms. The app goes beyond basic sketching — it includes a wide range of tools, shape drawing with live preview, emoji stamping, image filters, opacity control, and a complete undo/redo system with up to 20 steps.

How it's built

The app uses a Bitmap canvas with Graphics for all drawing operations. A temporary bitmap handles shape preview so users can see shapes before finalising them. Tools are managed through boolean flags (pen, brush, eraser, shapes, fill, text, emoji), and the undo/redo system uses two Stack<Bitmap> objects to store canvas states. Colour selection supports both standard and custom colour slots, with adjustable brush size and opacity via trackbars.
