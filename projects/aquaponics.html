Smart Aquaponics System

Automated sustainable farming system — 1st Prize, NYP TechNexus 2025.

Role
Coding · Sensor Calibration · Logic Design
Type
Team Project · IoT
Result
1st Prize — NYP TechNexus 2025
Stack
M5Stack · UIFlow2 · MQTT

About the project

I collaborated with my team to develop a Smart Aquaponics System that automates sustainable farming using fish tank waste as natural fertilizer. The system recycles water in a closed loop to nourish crops while reducing environmental impact.

How it's built

We used M5Stack modules and UIFlow2 to automate key parameters such as pH, EC, light, temperature, and water flow. I contributed to the coding, sensor calibration, and logic design to support multi-crop growth conditions. The prototype was built using low-cost materials, with adjustable covers and efficient control systems to simulate real-world smart farming environments.

Outcome

This project led our team to win First Prize at NYP's TechNexus event, in collaboration with industry partners — an experience that deepened my interest in IoT, environmental technology, and automation.
