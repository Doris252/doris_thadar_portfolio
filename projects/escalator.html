// Project 05

Escalator Automation System

Energy-saving escalator prototype that activates only when users are detected — built from scratch, earned a Distinction.

Goal
Save energy with sensor-triggered escalator
Type
Individual Project
Result
Distinction
Tools
IR Sensors · Servo Motor · Arduino · PyCharm

About the project

I designed an energy-saving escalator prototype that activates only when in use. Using IR sensors, an Arduino board, and PyCharm, I built the circuit from scratch and programmed the logic to detect motion and control a servo motor.

How it works

The project mimics real escalator behaviour and helps reduce standby energy consumption. When an IR sensor detects a person approaching, the servo motor activates the escalator steps. Once the exit sensor detects the person has passed through, the system powers back down — all handled through a custom action flowchart I designed before building.

Outcome

It was part of a school module and earned me a Distinction.
