PCB Design Project

Custom ATmega328P-based PCB with a 3-digit 7-segment display, designed and routed from scratch in Eagle CAD.

Type
Individual Project
Tool
Eagle CAD
Microcontroller
ATmega328P
Display
3-digit 7-segment · Button Inputs

About the project

I designed a custom PCB board featuring the ATmega328P microcontroller, intended to control a 3-digit 7-segment display with button inputs. The board includes essential components like resistors, capacitors, transistors, a USB connection, and a 10MHz crystal oscillator.

Design process

I created the schematic and routed the PCB layout using Eagle CAD, ensuring proper spacing, signal clarity, and logical flow. I also optimised the placement of pins, push buttons, and display connections for functionality and testing.

What I learned

This hands-on project strengthened my skills in circuit design, microcontroller programming, and PCB layout troubleshooting.
