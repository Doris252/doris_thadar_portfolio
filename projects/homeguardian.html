My contribution

I was responsible for the Intrusion Detection subsystem and the web dashboard. The intrusion system used a camera connected to a BeagleBone Black Wireless, running OpenCV-based person detection with SSD MobileNet to identify intruders and trigger Telegram alerts. The web dashboard was built with Flask and Socket.IO, displaying live sensor feeds and system status in real time.
