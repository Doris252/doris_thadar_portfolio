About the project

I developed DorNick, a user-friendly shopping app using CustomTkinter in Python. The application features an interactive shopping cart, real-time price calculations, and simulated payment processing — all with basic data persistence for storing card information.

How it's built

Built with a modular code structure, it includes dynamic UI updates, user-friendly controls, and strong input validation. This project showcases my foundational skills in Python programming and GUI development, and reflects my interest in building real-world applications that are both functional and intuitive.
