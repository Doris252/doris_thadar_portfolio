// Project 07

Food Stall Website

Responsive multi-page website for a Japanese food stall at NYP's South Canteen — built with HTML5, CSS3, and JavaScript.

Type
Individual Project · Web Development
Stack
HTML5 · CSS3 · JavaScript
Context
School Module · NYP South Canteen

About the project

I designed a responsive, multi-page website for a popular Japanese stall located in Nanyang Polytechnic's South Canteen. The site showcases the stall's best-selling rice bowls (Donburi), provides a clean and accessible menu layout, and encourages user engagement through intuitive navigation and visuals.

How it's built

Built with HTML5, CSS3, and basic JavaScript, the project features a responsive navigation bar and structured styling for clarity. I also implemented a simple sign-up form with basic form validation. The goal was to create a mobile-friendly interface while practising user-centric design and file organisation.

Outcome

It was part of a school module and helped sharpen my front-end development skills and web design confidence.
